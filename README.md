# FixIQ — Powered by OpenSRE

Intelligent Incident Fix Advisor that tells you:
- What services are impacted
- How urgent the fix is
- Who gets affected by the fix
- What happened before and how it was fixed

## Usage

```bash
fixiq analyze -i alert.json
fixiq analyze -i alert.json --verify
```
